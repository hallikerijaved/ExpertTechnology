# Expert Technology - CCTV & Security Solutions

![Expert Technology Logo](assets/favicon.svg)

A modern, responsive website showcasing Expert Technology's premium CCTV installations, IP CCTV systems, Video Door Phones (VDP), and Annual Maintenance Contract (AMC) services. Built with clean HTML, CSS, and JavaScript for optimal user experience.

## 🚀 Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations and intuitive navigation
- **Service Showcase**: Detailed presentation of CCTV installation, IP systems, VDP, and AMC services
- **Project Gallery**: Visual showcase of completed security installations
- **Contact Integration**: Easy-to-use contact forms and direct call-to-action buttons
- **Performance Optimized**: Fast loading with optimized images and efficient code

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for better accessibility and SEO
- **CSS3**: Custom styling with modern features like Flexbox and Grid
- **JavaScript**: Interactive elements and smooth scrolling navigation
- **Font Awesome**: Professional icons for enhanced visual appeal
- **Google Fonts**: Custom typography using the Outfit font family

## 📦 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/expert-technology-cctv.git
   cd expert-technology-cctv
   ```

2. **Open in browser:**
   - Simply open `index.html` in your preferred web browser
   - No server setup required for static viewing

## 🎯 Usage

- Navigate through the sections using the top navigation menu
- Explore services to understand offerings
- View project gallery for installation examples
- Use the contact form to get quotes or inquiries
- Mobile-friendly hamburger menu for smaller screens
